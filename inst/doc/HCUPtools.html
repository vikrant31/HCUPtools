Getting Started with HCUPtools

Vikrant Dev Rathore
2025-11-29

HCUPtools is an R package for accessing and working with publicly available resources from the Agency for Healthcare Research and Quality (AHRQ) Healthcare Cost and Utilization Project (HCUP). The package provides streamlined access to HCUP's Clinical Classifications Software Refined (CCSR) mapping files and Summary Trend Tables.

Installation

install.packages("HCUPtools")

Quick Start

Download CCSR Mapping Files

library(HCUPtools)

# Download latest diagnosis CCSR mapping
dx_map <- download_ccsr("diagnosis")

# Download specific version
dx_map <- download_ccsr("diagnosis", version = "v2026.1")

Map ICD-10 Codes

# Create sample data
patient_data <- data.frame(
  patient_id = 1:3,
  icd10_code = c("E11.9", "I10", "M79.3")
)

# Map codes to CCSR categories
mapped <- ccsr_map(
  data = patient_data,
  code_col = "icd10_code",
  map_df = dx_map
)

Key Features

  • Download CCSR mapping files directly from HCUP
  • Map ICD-10 codes to CCSR categories
  • Download HCUP Summary Trend Tables
  • Generate proper AHRQ/HCUP citations
